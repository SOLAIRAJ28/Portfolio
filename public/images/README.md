# Image Setup Guide

This portfolio needs the following images in the `public/images/` folder:

## Required Images

1. **Avatar/Profile Picture**
   - File: `avatar.jpg`
   - Recommended size: 500x500px
   - Format: JPG or PNG
   - Description: Your professional headshot

2. **Project Images**
   - Files: `project1.jpg`, `project2.jpg`, `project3.jpg`, `project4.jpg`
   - Recommended size: 1200x800px
   - Format: JPG or PNG
   - Description: Screenshots or mockups of your projects

## Optional Images

3. **Favicon**
   - File: `vite.svg` (already included by Vite)
   - You can replace this with your own favicon

## Where to Find Free Images

If you don't have project screenshots ready, you can use placeholder images from:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Or generate screenshots from your actual projects

## How to Add Images

1. Create the `public/images/` folder if it doesn't exist
2. Add your images to this folder
3. Update the image paths in `src/data/portfolio.js`

## Image Optimization Tips

- Compress images before uploading (use tools like TinyPNG or Squoosh)
- Use WebP format for better performance
- Ensure images are responsive and look good on all devices
